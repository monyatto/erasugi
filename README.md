# えらすぎ

## サービスのURL

## 概要
えらすぎというサービスは、気軽に褒めたり褒められないという問題を解決したい、 褒められたい人・褒めたい人向けの、 画像作成&共有サービスです。
ユーザーは 褒めてほしいことをシェアしてリアクションをもらうことができ、定型的なリアクションのみが備わっている事が特徴です。

## 技術スタック

```
cd erasugi

```
